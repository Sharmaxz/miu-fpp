# 🧹 Problem 3: Remove Array Duplicates

Write a program called `Prog4`.

------------------------------------------------------------------------

## 🎯 Objective

Create a program that:

1.  Accepts an **array of Strings**
2.  Creates a **new array**
3.  Removes all duplicate Strings from the original input array
4.  Does **NOT** use any APIs such as:
  -   `HashSet`
  -   `TreeSet`
  -   or any similar collection utilities
5.  Implements your **own logic**
6.  Ensures the resulting array:
  -   Contains **no duplicate values**
  -   Contains **no null values**

------------------------------------------------------------------------

## 📌 Example

### Input:

    ["horse", "dog", "cat", "horse", "dog"]

### Output:

    ["horse", "dog", "cat"]

------------------------------------------------------------------------

## ⚠️ Requirements Summary

-   Use only arrays
-   Do not use Java Collection APIs
-   Do not return null values
-   Preserve only unique elements in the final array
